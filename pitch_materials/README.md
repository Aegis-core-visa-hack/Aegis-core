## PPT Generation Instructions

### Slide Structure (12 slides total)

Refer to `pitch_deck.md` for complete content. Each slide section includes:
- **Content**: Text and layout
- **Visual Description**: What to show
- **Speaker Notes**: What to say

### Design Guidelines

**Color Palette:**
- Background: Dark Navy `#0A1628`
- Primary Accent: Cyan `#00D4FF`
- Secondary Accent: Purple `#8B5CF6`
- Alert Red: `#EF4444`
- Success Green: `#22C55E`
- Warning Amber: `#F59E0B`

**Typography:**
- Headings: Bold, large, white
- Body: Medium weight, light gray
- Code/Tech: Monospace font

**Style:**
- Dark mode throughout
- Glassmorphism effects on cards
- Minimal, professional
- Finance/enterprise aesthetic

### Image Placement

| Slide | Image File | Usage |
|-------|-----------|--------|
| 1 (Title) | `aegis_hero_*.png` | Full background |
| 2 (Problem) | `problem_visualization_*.png` | Left half of slide |
| 5 (Architecture) | `multi_agent_system_*.png` | Center, large |
| 8 (Dashboard) | `compliance_dashboard_*.png` | Full width |

### Key Slides to Emphasize

1. **Slide 2 (Problem)** - Make them feel the pain
2. **Slide 7 (Demo Walkthrough)** - Show concrete value
3. **Slide 8 (Dashboard)** - Visual proof of concept
4. **Slide 10 (Impact)** - ROI numbers

### Presentation Timing

- Total: 10 minutes (8 min presentation + 2 min Q&A)
- Spend most time on: Problem, Demo, Impact
- Quick through: Architecture details, Tech stack

## PPT Software Recommendations

**Option 1: Google Slides**
- Use dark theme template
- Import images directly
- Easy collaboration

**Option 2: PowerPoint**
- Use "Ion" or "Facet" dark theme
- Customize colors to match palette
- Export as PDF backup

**Option 3: Canva**
- Search "startup pitch deck dark"
- Customize with our content
- Professional templates available
